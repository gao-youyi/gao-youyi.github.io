博客内容规范化说明

本压缩包对原始 source 目录做了统一整理，主要包括：

  • 统一所有文章文件命名为 YYYY-MM-DD-slug.md；
  • 为缺失 front-matter 的文章补齐 title / date / description / categories / tags / toc；
  • 将单值分类统一改为数组写法；
  • 为文章统一补充摘要与 <!--more--> 自动折叠位置；
  • 为以代码开头的文章补充开头说明，改善首页展示效果；
  • 修复 categories/index.md 与 tags/index.md 的格式错误；
  • 优化 about/index.md 内容；
  • 对明显未完成的文章补充基础信息，并设置 published: false，避免误发布。
