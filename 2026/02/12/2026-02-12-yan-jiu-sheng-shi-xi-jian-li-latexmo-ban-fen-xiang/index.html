研究生实习简历 LaTeX 模板分享：一页纸就够，别把简历写成论文

发表于2026-02-12|更新于2026-03-08|求职准备
|总字数:1.7k|阅读时长:5分钟|浏览量:

最近在重新整理实习简历，也顺手把自己的 LaTeX 模板又改了一版。

说实话，我一直挺喜欢用 LaTeX 写简历，不是因为它多高级，而是因为它对版式控制更稳定，尤其对我这种会反复改字段顺序、字间距、项目 bullet 的人来说，比 Word 舒服很多。

但简历这东西，模板永远只是第二位。真正重要的还是内容取舍。

所以这篇我不只放模板思路，也会一起讲讲我现在怎么理解“研究生实习简历”这件事。

1. 研究生实习简历最重要的，不是写全，而是写准

很多同学一到研究生，就容易把简历越写越满：

  • 本科经历舍不得删；
  • 研究生科研细节写很长；
  • 课程、技能、奖项全部堆满；
  • 一页放不下就想做两页。

但如果你投的是实习，尤其是算法岗、开发岗、研究型实习，我现在还是建议：

一页纸优先。

因为面试官筛简历时，最想先看到的是：

  • 你是谁；
  • 你做过什么最有代表性的事；
  • 你和岗位匹不匹配。

不是来看你的成长史。

2. 我现在更推荐的一页结构

我自己的简历大概会按下面顺序排：

  1. 基本信息
  2. 教育背景
  3. 科研经历 / 项目经历
  4. 实习经历（如果有）
  5. 竞赛与荣誉
  6. 技能补充

如果你偏研究，就把科研放前面。

如果你偏开发，就把项目和实习放前面。

顺序一定要服务目标岗位，而不是一成不变。

3. 我为什么一直喜欢用 LaTeX

主要是几个原因：

1. 格式稳定

不会像 Word 那样改一行，下面全乱。

2. 更适合一页压缩

字号、行距、列表间距都能精细控制。

3. 适合技术类简历

尤其是你要放代码仓库、论文、项目名、技术栈时，LaTeX 的整体观感通常更干净。

当然，LaTeX 也不是没缺点。最大的缺点就是：

  • 上手门槛更高；
  • 改模板一开始很烦；
  • 中文环境容易出点小问题。

但只要你有一个可复用的模板，后面维护成本其实不高。

4. 我自己的模板思路

我现在的模板思路比较简单：

  • 顶部放姓名、邮箱、电话、主页 / GitHub；
  • 中间按模块展开；
  • 每条经历尽量控制在 2～3 个 bullet；
  • 每个 bullet 只写最有信息密度的内容。

这只是骨架，不是最终版。真正的重点还是后面怎么填内容。

5. 简历 bullet 我现在怎么写

我越来越不喜欢那种特别空的表达，比如：

负责项目开发，提升了团队协作能力。

这种话没有任何竞争力。

我更推荐一种结构：

问题背景 + 你的动作 + 结果 / 指标

比如：

  • 面向传感器噪声与随机缺失场景，设计双路径时空建模框架，并完成多组消融实验；
  • 独立完成基于 NestJS、Prisma、PostgreSQL 的 Web 服务搭建，实现 Admin 登录、RBAC 与审计日志；
  • 将裁决结果标准化为结构化 JSON，结合缓存与限条数机制降低 LLM 调用成本。

这样一句话里，信息密度会高很多。

6. 研究生实习简历和保研简历的区别

这两者很像，但重点不一样。

保研简历更看“潜力”和“培养价值”

老师会在意：

  • 排名
  • 英语
  • 科研潜力
  • 竞赛和项目基础

实习简历更看“能不能用”

面试官更在意：

  • 你做过的东西和岗位是否相关；
  • 你会不会常见技术栈；
  • 你能不能很快上手；
  • 你有没有结果导向的经历。

所以到了研究生阶段，本科时期那些比较泛的经历，该删就删。

7. 我觉得最容易踩的坑

1. 技能写太多

会一点就写，很容易被追问到崩。

2. 项目写成流水账

没有重点，看不出你到底负责了什么。

3. 排版太满

一页纸不是越挤越强，留白真的很重要。

4. 文件命名混乱

建议直接：

5. 导出 PDF 不检查

尤其是中文、超链接、页边距、字体嵌入，有时候本地看着没问题，发出去就出事。

8. 如果你现在就要开始改模板，我建议这样做

第一步：先写内容，不急着美化

内容没定下来，模板改再久都没意义。

第二步：把所有经历压成结果导向 bullet

每条经历尽量控制在 2～3 条内。

第三步：再做排版压缩

字体、间距、模块顺序，这时候再调效率最高。

第四步：找人帮你看

尤其找已经实习过、找过工作的同学看，反馈通常很直接。

最后

我现在对简历这件事的理解其实越来越朴素：

它不是作品集，也不是散文集，更不是“我很努力”的证明书。

它只是一个非常短的销售页面，目标就是让别人快速判断：

这个人，值不值得约来聊一轮。

模板当然重要，但模板真正的价值，是帮你把内容更稳定、更克制、更专业地呈现出来。

如果你刚好也在改研究生实习简历，希望这篇能帮你少走点弯路。
